Klassen

Schreiben Sie eine Klasse Student mit folgenden Eigenschaften:

  • Ein neues Studenten-Objekt wird erstellt, indem man dem Konstruktor (d.h. der init Funktion Namen und Vornamen des Studenten übergibt:
student = Student(name="Müller", vorname="Heinz")

  • Die Klasse soll eine Objektvariable noten_liste besitzen, die zu Beginn eine leere Liste ist, sowie eine Methode neue_note, die der noten_liste eine neue Note hinzufügt

  • Schreiben Sie eine Methode durchschnitts_note, die den Durchschnitt aller Noten in noten_liste bildet
