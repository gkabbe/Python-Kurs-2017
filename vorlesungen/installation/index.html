0.1 Installation

Gabriel Kabbe

2017-03-27 16:31

Source

Python installieren

Diese Anleitung ist für alle gedacht, die sich schon im Voraus Python auf ihrem privaten Rechner installieren möchten. Eine ausführliche Erklärung bekommt ihr sowieso nochmal von mir, sobald der Kurs beginnt.

Um für diesen Kurs gewappnet zu sein, empfehle ich die Installation von Python via Anaconda. Dafür einfach auf den obigen Link drücken und auf der Webseite das Betriebssystem auswählen. Dann den Installer der Version 3.6 herunterladen. Wem der Anaconda Installer zu groß ist (~500 MB), der kann auch Miniconda installieren. Hier müssen einige der Pythonpakete, die wir später verwenden werden, allerdings nachinstalliert werden.
